

# Git File Stages and Basic Commands

## 1. Git File Stages

In Git, files move through different stages.

- **U — Untracked**  
  The file is new and Git is not tracking it.

- **A — Added / Staged**  
  The file has been added to the staging area and is ready to commit.

- **C — Committed**  
  The file is saved in the Git repository as a commit.
